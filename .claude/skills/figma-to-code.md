# Figmaデザインをコードに変換

Figma MCPを使ってデザインを取得し、HTML/CSSコードに変換します。

## 手順

1. **デザイン情報の取得**
   - ユーザーからFigma URLまたはノードIDが提供された場合: `get_design_context` でデザイン情報を取得
   - URLが不明な場合: 「FigmaのURLまたはページ名を教えてください」と確認
   - 必要に応じて `get_screenshot` でビジュアルを確認

2. **デザイン分析**
   - レイアウト構造（グリッド、フレックスなど）
   - カラーパレット
   - タイポグラフィ（フォント、サイズ、ウェイト）
   - スペーシング（マージン、パディング）
   - レスポンシブ対応のブレークポイント

3. **コード生成**
   - セマンティックHTMLで構造を作成
   - CSS変数を活用してカラーやスペーシングを定義
   - レスポンシブ対応（モバイルファースト）
   - アクセシビリティ対応（alt属性、ARIA属性）

4. **プレビュー確認**
   - 生成したコードをプレビューサーバー（`preview_start` で `website` を起動）で確認
   - Figmaデザインとの差異がないか確認

5. **ユーザー確認**
   - 生成結果を提示し、修正点がないか確認
   - 必要に応じて調整

## コーディング規約
- Figmaデザインに忠実であること
- レスポンシブ対応必須
- セマンティックHTML
- CSS変数でテーマ管理
- 画像はalt属性必須

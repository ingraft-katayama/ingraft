# SNS投稿原稿の作成

各プラットフォーム向けの投稿原稿を作成し、人間のレビューに回します。

## 手順

1. **要件の確認**
   - 対象プラットフォーム（X / Instagram / YouTube / 複数選択可）
   - 投稿テーマ・内容
   - 希望するトーン（カジュアル / フォーマル / etc.）
   - 投稿予定日時
   - 画像・動画の有無

2. **テンプレート確認**
   - `sns/templates/` にプラットフォーム別テンプレートがあれば参照
   - ブランドトーンを維持

3. **原稿作成**
   - プラットフォームごとの文字数制限・特性に合わせて作成:
     - **X**: 140文字以内、ハッシュタグ2〜3個
     - **Instagram**: キャプション2200文字以内、ハッシュタグ10〜15個、改行で読みやすく
     - **YouTube**: タイトル100文字以内、説明文5000文字以内、タグ設定
   - 複数パターンが求められた場合は3パターン提示

4. **下書き保存**
   - `sns/drafts/` に以下のフォーマットで保存:
     - ファイル名: `YYYY-MM-DD_{platform}_{topic}.md`
     - 内容: プラットフォーム、投稿日時、本文、ハッシュタグ、備考

5. **レビュー依頼**
   - 下書きの内容をチャットに表示
   - Slack MCPが利用可能であれば、指定チャンネルに通知
   - 「**投稿前に必ず人間が確認してください**」と明記

## 重要ルール
- **投稿は人間の承認なしに実行しない**
- 機密情報や個人情報が含まれていないか確認
- 誹謗中傷や炎上リスクのある表現を避ける
- 競合他社への直接的な言及を避ける

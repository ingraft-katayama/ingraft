# 動画編集（FFmpegベース）

台本や指示に基づいてFFmpegコマンドで動画を編集します。

## 前提条件

FFmpegがインストールされていること。未インストールの場合:
```bash
brew install ffmpeg
```
`ffmpeg -version` で確認。

## 手順

1. **要件確認**
   - 編集内容を確認:
     - トリミング（開始時間〜終了時間）
     - 複数クリップの結合
     - テロップ（字幕）追加
     - ウォーターマーク追加
     - ショート動画への切り出し（縦動画 9:16）
     - 音声調整
   - 素材ファイルのパスを確認
   - 台本がある場合はその内容を読み込む（Google Drive上のファイルも可）

2. **素材の確認**
   - `ffprobe` で素材の情報を取得（解像度、コーデック、長さ等）
   - 素材が複数ある場合はリスト化して確認

3. **FFmpegコマンド生成・実行**
   - 要件に応じたFFmpegコマンドを生成
   - コマンドの説明をユーザーに提示
   - 確認後に実行

   よく使うパターン:
   - **トリミング**: `ffmpeg -i input.mp4 -ss HH:MM:SS -to HH:MM:SS -c copy output.mp4`
   - **結合**: concat demuxerを使用
   - **テロップ**: `drawtext` フィルタまたは `.srt` ファイルでの字幕焼き込み
   - **ショート切り出し**: `crop` + `scale` で 1080x1920 に変換
   - **ウォーターマーク**: `overlay` フィルタ

4. **出力確認**
   - `ffprobe` で出力ファイルの情報を確認（解像度、長さ、ファイルサイズ）
   - サムネイル画像を生成して確認: `ffmpeg -i output.mp4 -ss 00:00:05 -frames:v 1 thumbnail.jpg`

5. **修正対応**
   - ユーザーが映像を確認した後の修正指示に対応
   - テロップの文言修正、タイミング調整等

## ショート動画への切り出し

ショート動画（60秒以内、9:16縦動画）の切り出し手順:
1. 元動画からハイライト区間を指定（ユーザーに確認）
2. 縦動画（1080x1920）にクロップ・リサイズ
3. 必要に応じてテロップを追加
4. 出力ファイルは `video/exports/` に保存

## 出力先
- 編集済み動画: プロジェクトルートまたはユーザー指定のパス
- ショート動画: `video/exports/`（.gitignoreに含まれるため手動管理）

## 注意事項
- 大容量ファイルはGitにコミットしない
- エンコードに時間がかかる場合はその旨を伝える
- 品質とファイルサイズのトレードオフをユーザーに説明

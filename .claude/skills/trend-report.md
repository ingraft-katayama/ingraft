# SNSトレンドレポート生成

Web検索やSNS APIを使ってトレンドを収集し、日本語レポートを作成します。

## 手順

1. **トレンド収集**
   - Web検索で以下のカテゴリのトレンドを調査:
     - デジタルクリエイティブ・デザイン業界
     - テクノロジー・Web開発
     - SNSマーケティング
     - ingraft inc. の事業領域に関連するトピック
   - X API が利用可能な場合、日本のトレンドトピックも取得

2. **競合・業界分析**
   - 同業他社のSNS投稿傾向（公開情報のみ）
   - 業界で話題になっているキーワード
   - バズっているコンテンツの形式・テーマ

3. **レポート作成**
   以下のフォーマットで日本語レポートを作成:

   ```
   # トレンドレポート YYYY-MM-DD

   ## 今週の注目トレンド
   1. [トレンド1]: 概要と背景
   2. [トレンド2]: 概要と背景
   3. [トレンド3]: 概要と背景

   ## コンテンツアイデア
   - トレンドを活用した投稿案1
   - トレンドを活用した投稿案2
   - トレンドを活用した投稿案3

   ## 推奨アクション
   - 今週中に対応すべきこと
   - 来週以降の計画
   ```

4. **共有**
   - レポートをチャットに表示
   - Slack MCPが利用可能であれば、指定チャンネルに投稿

## データソース
- Web検索（常に利用可能）
- X API（環境変数 `TWITTER_API_KEY` が設定されている場合）
- Google Trends（Web検索経由）
